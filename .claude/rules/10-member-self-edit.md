# Member Self-Edit Rules

## 자연어 수정 기능

각 멤버는 자신의 역할 파일(`docs/members/{name}.md`)을 자연어로 수정할 수 있다.

### 수정 가능 범위
- 환영 메시지 변경
- 담당 영역 추가/제거
- PR 체크리스트 항목 추가/제거
- Claude Code 지침 수정
- 코딩 스타일 규칙 추가

### 수정 트리거 예시
```
"내 담당 영역에 {영역} 추가해줘"
"체크리스트에 {항목} 추가해줘"
"내 환영 메시지 바꿔줘"
"내 지침 {N}번 수정해줘"
```

### 보안 규칙
1. **본인 파일만 수정 가능**: Justin은 justin.md만, Kevin은 kevin.md만
2. **다른 멤버 파일**: 읽기만 가능, 수정은 해당 멤버에게 요청
3. **README.md**: Hoon만 수정 가능 (권한 매트릭스 포함)
4. **수정 후**: 파일 하단 변경 이력에 기록

### Claude의 검증
1. 현재 세션 멤버 확인
2. 요청된 파일이 본인 파일인지 검증
3. 본인 파일이 아니면: "이 파일은 {멤버}의 역할 파일입니다. {멤버}에게 수정을 요청하시겠습니까?"
